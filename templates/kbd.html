Keyboard Layout

I created Windows and Ubuntu keyboard layouts so you can try typing in Phonetic English.

The Windows download contains "setup.exe", which installs the "Inggliš (Funetik)" keyboard layout in Windows. Once installed, press Ctrl+Shift to switch between layouts. It also contains "ingfun.klc", the keyboard layout file for Microsoft Keyboard Layout Creator.

The Ubuntu download contains the file "ing", which should be placed in /usr/share/X11/xkb/symbols/, and "README", which includes instructions for setting up the layout. Once installed, press Super+Space to switch between layouts.

Download for Windows: ingfun-kbd-windows.zip

Download for Ubuntu: ingfun-kbd-ubuntu.tar.gz

Demo Keyboard

You can get an idea of what the keyboard layout looks like below. Click the Shift key on the keyboard below or hold down on your keyboard's actual shift key to see the alternate letters.

You can also see the history of the phonetic keyboard by clicking on the version links below it.

The dark gray keys are the ones that made it to the final version, while the lighter gray keys are ones that were just used in an earlier version.

Beneath the keyboard is a sentence I wrote that uses each simplified phoneme of American English, to demonstrate what each orthography looks like.
